# DATA-CRM

Proyek ini adalah sistem **Customer Relationship Management (CRM)** yang digunakan untuk mengelola data pelanggan.

## Cara Menggunakan
- Buka file `cekdatacrmindex.html` untuk memulai.
- Ikuti langkah-langkah yang tertera di halaman.

## Fitur
- Mengelola data pelanggan.
- Melakukan analisis data CRM.

## Lisensi
- Proyek ini bersifat open-source.
